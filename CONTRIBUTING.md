# Contributing to awesome-shizuku

Want to add an app or tool to this list? Follow these guidelines:

## Format
```markdown
- **App Name** ([link](url)) — Brief description. No root required.
```

## What we accept
- ✅ Shizuku-powered Android apps
- ✅ Legitimate security/privacy tools
- ✅ Active projects (updated in last 6 months)
- ✅ Open source preferred

## What we don't accept
- ❌ Abandoned/unmaintained projects
- ❌ Closed-source apps
- ❌ Spyware or malicious tools
- ❌ Spam or self-promotion without value

Submit a pull request with your addition!
