# 🛠️ Shizuku-Powered Tools

Tools and apps that use Shizuku for ADB-level access without root.

## System Tools

| Tool | Function | GitHub |
|------|----------|--------|
| **App Manager** | Full app control — perms, ops, components | [muntashirakon](https://github.com/muntashirakon/AppManager) |
| **Bloatware Remover** | Remove system apps via ADB | [1hakr](https://github.com/1hakr/Bloatware-Remover) |
| **Shizuku-Next** | Shizuku fork with extra features | [RikkaW/Shizuku](https://github.com/RikkaW/Shizuku) |
| **NetGuard** | Per-app firewall | [M66B/NetGuard](https://github.com/M66B/NetGuard) |
